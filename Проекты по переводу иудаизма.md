Библия: от текстуальной критики до мультиязычных корпусов

- **📝 Текстологическая критика (BibCrit)**  
    Сравнивает и анализирует древние версии Библии (Масоретский текст, Септуагинту, Свитки Мертвого моря)[](https://github.com/Jossifresben/BibCrit). Он сопоставляет фрагменты, оценивая их расхождения[](https://github.com/Jossifresben/BibCrit), и выявляет поздние теологические наслоения[](https://github.com/Jossifresben/BibCrit).
    
- **🗣️ Переводы на редкие языки (eBible Corpus)**  
    Огромный корпус из 891 языка[](https://huggingface.co/datasets/DavidCBaines/ebible_corpus) выравнивает переводы Библии на уровне стихов для обработки естественного языка[](https://huggingface.co/datasets/DavidCBaines/ebible_corpus).
    
- **🖋️ Анализ и идентификация писцов (Enoch)**  
    Модель на основе нейросети (BiNet) для датировки манускриптов с точностью около 30 лет[](https://erc.europa.eu/news-events/news/unlocking-timecode-dead-sea-scrolls), а также идентификации рук неизвестных переписчиков и определения количества авторов.
    
- **📚 Мультиязычные параллельные корпуса**
    
    - **MaSS**: Восемь языков и аудиосегменты для развития систем распознавания речи (ASR) и Text-to-Speech.
        
    - **AG-MG Parallel Corpus**: 132,000 предложений для перевода древнегреческого на современный греческий.
        
    - **Faith CLI**: Инструмент для автоматизированного извлечения данных из Библии в формате JSON.
        
- **✡️ Уникальные языковые пары (иврит-арамейский)**  
    Модели MarianMT для прямого перевода между ивритом и арамейским языком таргумов (например, для перевода на Таргум Онкелос)[](https://github.com/johnlockejrr/johnlockejrr)[](https://huggingface.co/johnlockejrr/marianmt-he2arc-targum-voc).

### ✡️ Танах / Тора: от религиозных приложений до анализа смыслов

- **📱 OHRBIT (Orthodox Union)**  
    Бесплатное приложение для персонализированного изучения Торы, подбирающее материалы по уровню и интересам пользователя.
    
- **📑 Анализ стилей переводов (Septuagint)**  
    Анализирует Септуагинту, сравнивая тематику и настроение (сентимент-анализ) на основе перевода Вульгаты и английских версий[](https://digital.kenyon.edu/dh_iphs_ss/40/).
    
- **⚖️ Количественная оценка переводов**  
    Автоматически сравнивает 30 английских переводов Библии и группирует их, используя технологию USD и специальную математическую метрику[](https://m2.mtmt.hu/api/publication/35133496?&labelLang=eng).
    
- **💬 Интеграция в современные системы**  
    Пакет WideHoly для AI-ассистентов (MCP сервер), который позволяет производить поиск по межрелигиозным писаниям и сопоставлять стихи для Торы, Библии и Корана.

### 📚 Талмуд: решение сложной задачи машинного перевода

- **🏛️ Национальный исследовательский совет Италии (CNR)**  
    В рамках итальянского проекта перевода Вавилонского Талмуда используют LLM вместе с памятью переводов и глоссариями для повышения единообразия[](https://iris.cnr.it/handle/20.500.14243/562386?mode=full). RAG-метод улучшает ответы: в тестах предложения были лучше прямых машинных в 79% случаев[](https://iris.cnr.it/handle/20.500.14243/562386?mode=full).

### 💎 Самое нишевое в библейском ИИ

- **🌟 Нишевые проекты и факты**
    
    1. **Реконструкция древнееврейского оригинала из греческого**: Модуль `Back-Translation Workbench` на основе методологии Това по слову восстанавливает исходный текст[](https://github.com/Jossifresben/BibCrit).
        
    2. **Обнаружение литературных приемов**: ИИ ищет в тексте хиазмы, инклюзивы и параллельные панели[](https://github.com/Jossifresben/BibCrit).
        
    3. **Моделирование «смешанного происхождения»**: Анализирует источники в Пятикнижии, распределяя стихи между разными традициями[](https://github.com/Jossifresben/BibCrit).
        
    4. **Канон на службе у лингвистики**: Библия уникальна для создания NLP-датасетов на сотнях языков благодаря единой структуре.
        
    5. **Трудности перевода Талмуда**: LLM не понимают обязательные разъяснительные вставки редактора[](https://iris.cnr.it/handle/20.500.14243/562386?mode=full).

#социум #синестезия #оргазм #биология 


| Параметр | Линейный (генитальный) | Нелинейный (топ-даун / альтернативный) |
| ----------------------- | ------------------------------- | --------------------------------------------------------- |
| **Источник сигнала** | Гениталии → спинной мозг → кора | Кора (фантазия/дыхание) → тело через вегетативную систему |
| **Главный гормон** | Дофамин (пиковый выброс) | Окситоцин + эндорфины (волнообразно) |
| **Длительность** | Короткая (3–10 с) | Длинная (13–50 с, может повторяться волнами) |
| **Рефрактерный период** | Есть (падение пролактина) | Почти отсутствует (пролактин минимален) |
| **Ощущения** | Локальный взрыв в тазу | Разлитое тепло, изменение сознания, потеря границ тела |
| **Требуемые навыки** | Механическая стимуляция | Осознанность, управление дыханием, фантазия |
